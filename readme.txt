1. Чат на сокетах.
2. Реализовал миниму, потому что времени совсем нет, к сожалению.
3. На эту базу можно уже накрутить норм авторизацию, комнаты, список пользователей, информирование о входе в чат и выходе из него и т.д. - все упирается только во время.
4. На этом простом примере постарался показать возможности использования нескольких компонентов, роутинга, сокетов, директив. 
5. Чтобы запустить связку и проверить работу необходимо с помощью консоли перейти в папки front и back, а затем выполнить команду npm i - установит все необходимые модули для работы.
6. Запуск Front-а: перейти в каталог front и в консоли выполнить команду ng serve
7. Запуск Back-а: перейти в каталог back и в консоли выполнить команду node backend.js
